À propos de l'importation des testaments

Sources et ordre de saisi

Les champs sourcés sont des champs pour lesquels il est possible d'indiquer plusieurs sources pour une même valeur.
C'est utile - par exemple - si la mention de décès d'un testateur est contradictoire entre sa fiche Mémoire des Hommes et l'Etat civil.
Dans cette situation, vous pouvez écrire les valeurs de la manière suivante : valeur 1 [source 1] | valeur 2 [source 2]. Les sources sont entre crochets, les valeurs sont séparées par une barre verticale.
Les sources possibles sont :

  • MDH : pour Mémoire des Hommes
  • NOT : pour minute notariale
  • TES : pour testateur
  • EC : pour Etat civil
  • AS : pour Autre source

Pour chaque champ (date de décès, métier, nom, etc.), un régime de priorité a été défini pour ces sources. Pour en savoir plus, consultez le tableau récapitulatif du tableau de saisi des métadonnées.
